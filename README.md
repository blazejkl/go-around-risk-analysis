# Aircraft Go-Around Analysis

Exploratory data analysis project focused on aircraft go-around events using OpenSky flight data and METAR weather variables.

The analysis investigates how go-around frequency changes across time, aircraft types, airlines and weather conditions such as visibility and wind speed.

## Technologies
- Python
- pandas
- NumPy
- Matplotlib
- scikit-learn
- ipywidgets
- Jupyter Notebook

## Scope
- data cleaning and preprocessing
- feature engineering for time, visibility and wind speed
- exploratory analysis of go-around frequency and ratios
- visual analysis by hour, month, aircraft type, airline and weather conditions
- simple logistic regression model estimating go-around probability
- interactive Jupyter widget for changing wind and visibility inputs

## Data
The project uses OpenSky aviation data and METAR weather variables. The source dataset was based on publicly available OpenSky data prepared for research on aircraft go-around trajectories.
